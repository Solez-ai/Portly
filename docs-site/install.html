CLI Install

Portly supports direct usage via npx or optional global installation.

Prerequisites

Node.js

Node 18+ is recommended for current dependencies and runtime behavior.

NPM Access

Install from npm registry and run from any terminal environment.

Running App

Start your local dev server before executing Portly auto-detection mode.

Quick Start

npx @solez-ai/portly

Auto-detect mode scans common development ports and opens the first active target.

Global Install

npm install -g @solez-ai/portly
portly

Useful for daily use without prefixing with npx.

Project Setup (Local Development)

git clone https://github.com/Solez-ai/Portly.git
cd portly
npm install
npm start

Use this setup when contributing code, editing docs, or preparing package releases.

Serve Documentation Site

npm run docs:serve

Runs static docs site locally at http://localhost:4173.
