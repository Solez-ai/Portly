CLI Install

Install Portly from npm and use the command blocks below with one-click copy and platform tabs.

Quick Start

npx @solez-ai/portly
npm install -g @solez-ai/portly
portly doctor
npx @solez-ai/portly
npm install -g @solez-ai/portly
portly doctor

Project Setup (Local Development)

git clone https://github.com/Solez-ai/Portly.git
cd Portly/portly
npm install
npm run share

Command Presets

dev preset

portly dev

demo preset

portly demo

watch mode

portly watch

Extra Setup Helpers

npm run hooks:install
npm run docs:serve
portly list
portly stop

hooks:install creates a free local pre-push hook for lint + doctor checks.
