Examples

Real command patterns for development teams, demos, QA sessions, and stakeholder sharing.

Auto Detect Local Dev Server

npx @solez-ai/portly

Best for Vite, Next.js, and common local stacks using known default ports.

Tunnel a Specific Port

npx @solez-ai/portly 3000

Bypass detection and expose the exact service port you select.

Request a Named Session

npx @solez-ai/portly 3000 --name samin

Request stable branding for demos and easier verbal sharing during calls.

Branded Display Domain

npx @solez-ai/portly

Use this only when you own wildcard DNS/routing for that host. Otherwise, use the provider URL output.
